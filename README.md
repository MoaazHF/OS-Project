# OS Scheduler Comparison – Round Robin vs Shortest Job First

A GUI‑based comparison of **Round Robin** and **Shortest Job First (SJF)** scheduling algorithms built with Python and Tkinter.

## Features
- Add/remove processes dynamically
- Validate all inputs (quantum, burst times, arrivals)
- Run both algorithms side‑by‑side and view:
  - Gantt charts
  - Detailed metrics (CT, TAT, WT, RT) with averages
  - Ready queue visualisation
  - Automatic comparison summary and conclusion
- Five built‑in test scenarios (A–E) including a validation case
- Plain procedural code – no OOP, easy to read and modify
